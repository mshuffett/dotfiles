║ WORKFLOW TIPS                                                                                                                                         ║
╠═══════════════════════════════════════════════════════════════════════════════════════════════════════════════════════════════════════════════════════╣
║  • Create project sessions: tn my-project                                                                                                             ║
║  • Each session has independent window layouts                                                                                                        ║
║  • Use Ctrl+h/j/k/l everywhere for seamless navigation between nvim splits and tmux panes                                                             ║
║  • Quick pane jumping: Opt+1 through Opt+9 to jump directly to panes (Opt = Alt = Option key on Mac)                                                 ║
║  • Quick window switching: Opt+h/l for previous/next window without prefix (faster than Ctrl+a n/p)                                                   ║
║  • Zoom pane for focus: Ctrl+a z (perfect for full-screen coding, then unzoom to see other panes)                                                     ║
║  • Statusline indicators: ● = prefix active, ○ = prefix inactive, 🔍 ZOOM = pane is zoomed                                                            ║
║  • tmux-fingers quick copy: Ctrl+a f shows hints on paths/URLs/hashes/IPs - type hint letter to copy, Shift for alt action, Tab for multi-select     ║
║  • tmux-open from selection: highlight text, then Ctrl+a o (open) or Ctrl+a Ctrl-o (edit) or Ctrl+a Shift-s (search)                                ║
║  • Unified search: Ctrl+a Ctrl+f searches ALL sessions/windows/panes in one list. Type [SESSION]/[WINDOW]/[PANE] to filter by type                 ║
║  • Direct finders: Ctrl+a s (sessions), Ctrl+a w (windows), Ctrl+a M (join pane) - jump straight to that category                                 ║
║  • Use 'q' or any key to close this cheatsheet, or navigate with less commands (↓/↑, j/k, PgDn/PgUp, /, n)                                           ║

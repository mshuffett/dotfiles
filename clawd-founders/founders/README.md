# Founders Directory

Store one markdown file per founder: `<name>.md`

## Template

```markdown
# [Founder Name]

## Contact
- Phone: +1XXXXXXXXXX
- Company: [Company Name]
- Role: [CEO/CTO/etc]
- Email: [optional]

## Context
[Brief background, what they're working on, stage]

## Interaction History
### YYYY-MM-DD
- [Summary of interaction]

## Follow-ups
- [ ] [Pending action items]

## Notes
[Anything useful to remember]
```

## Tips
- Keep interaction history concise
- Update after every meaningful exchange
- Use follow-ups for actionable items

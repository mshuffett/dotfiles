When drafting a forwardable intro/reconnect blurb for Michael (investor/partner outreach), the shape he converged on (BCV/Christina, Jun 2026):

- **Write it in the connector's voice, ready to send verbatim** — open with "<Name> —" and end with "want me to intro you three?" so the middle-person can paste it without edits and isn't put on the hook to own a meeting.
- **Lead with the real relationship** — reconnect framing ("you met them briefly at <event>"), not a cold pitch. If they've met, it's a re-intro.
- **Keep it about the entity (Open Batch), not Michael's personal raise** — unless he explicitly says to include the raise. The value prop = "connect with the batch" (meet the founders, join demo day).
- **One concrete hook, no hype** — e.g., "founders, many raising now"; cut salesy phrases ("high-signal," "curated funnel") and redundancy ("repeat YC founders" already implies "experienced second-time").
- **Tight + skimmable** — ~3-4 sentences: reconnect line · what-it-is line · the ask. Soft close.
- **Co-run framing:** Open Batch is run by **Michael Shuffett and Dave Chen** together.

**Why:** he iterates these a lot for the raise; this is the form that survived a 3-lens review (VC recipient / connector / editor).
**How to apply:** default to this when he asks for an intro blurb; the per-deal facts (who, what event, what firm) live in the CRM, not here. The artifact (final blurb) goes on the relevant CRM row, not in memory.
